# RPGgame
This is a simple RPG game made by Charles and Nick. This is an active work in progress!

https://postimg.org/image/4icyhl47t/

You can run the game by running the main method within Launcher.java.

Controls:

Movement - W,A,S,D
Talk - Enter
Battle - Space
Quit - Escape
Mute - M
Menu - P
